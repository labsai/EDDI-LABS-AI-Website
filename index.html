Why E.D.D.I?

Digitization of workflows and human communication

Open Source

EDDI ist Open Source, developed in Java, provided with Docker and can thus be used in any modern cloud environment

Easy configuration

The code is self-explanatory and easy to extend. The configuration is open and flexible.

Skills in kit form

Ready-to-use algorithms for optimal automation of human communication.

Red Hat certified

The only Open Source Chatbot Technology that is Red Hat OpenShift certified and available on the Red Hat marketplace.
