  <div class="text-center mb-12">
    <h2 class="text-3xl font-bold text-white! border-0! text-center">How It Works</h2>
  </div>

```
1. Create a trigger: intent="billing" → agentId="billing-agent"
2. User sends: chat_managed(intent="billing", userId="user1", message="What's my balance?")
3. EDDI:
   a. Looks up trigger for "billing" intent
   b. Finds or creates a conversation for userId="user1" + intent="billing"
   c. Routes the message to the billing-agent
   d. Returns the response
4. Next message from same user + intent reuses the same conversation
```
