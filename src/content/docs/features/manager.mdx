  <div class="text-center mb-12">
    <h2 class="text-3xl font-bold text-white! border-0! text-center">Frequently Asked Questions</h2>
  </div>

<details>
<summary><strong>Is the Manager included in the open-source version?</strong></summary>

Yes. The EDDI Manager is fully open-source under the same Apache 2.0 license as the EDDI backend. There is no "community vs. enterprise" split — all features are available in the open-source version.

</details>

<details>
<summary><strong>What tech stack is the Manager built on?</strong></summary>

The EDDI Manager is a **React** single-page application built with modern tooling: React Router for navigation, i18next for internationalization, and a component library optimized for the EDDI API surface. It communicates with EDDI via the standard REST API.

</details>

<details>
<summary><strong>Can I still use the REST API and MCP tools directly?</strong></summary>

Absolutely. The Manager is one of three interfaces to EDDI — alongside the **REST API** (for programmatic integration) and the **MCP server** (for AI-native control via Claude Desktop, Cursor, etc.). All three interfaces are fully interchangeable; changes made in the Manager are immediately visible via API and vice versa.

</details>

<details>
<summary><strong>Which languages does the Manager support?</strong></summary>

The Manager supports **11 locales**: English, German, French, Spanish, Italian, Portuguese, Dutch, Polish, Czech, Turkish, and Arabic (with full RTL layout support). All UI elements, tooltips, error messages, and onboarding tours are fully translated.

</details>
</section>
