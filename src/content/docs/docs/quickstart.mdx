---
title: Quickstart
description: Get EDDI running and create your first agent in under 5 minutes.
---

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Start EDDI**

   ```bash
   docker run -p 7070:7070 labsai/eddi:latest
   ```

2. **Create an agent** via the MCP tool or REST API:

   ```bash
   curl -X POST http://localhost:7070/mcp \
     -H "Content-Type: application/json" \
     -d '{"method": "tools/call", "params": {"name": "setup_agent", "arguments": {"name": "Quick Agent", "systemPrompt": "You are a helpful assistant.", "provider": "anthropic", "modelId": "claude-sonnet-4-6"}}}'
   ```

3. **Chat** with your agent:

   ```bash
   curl -X POST http://localhost:7070/mcp \
     -H "Content-Type: application/json" \
     -d '{"method": "tools/call", "params": {"name": "chat_with_agent", "arguments": {"agentId": "<your-agent-id>", "message": "Hello!"}}}'
   ```

4. **Open the Manager UI** at `http://localhost:7070` to visually manage your agents.

</Steps>
