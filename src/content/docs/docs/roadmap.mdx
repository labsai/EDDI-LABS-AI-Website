## v6.0 — Current Release ✅

The v6.0 release represents a major architectural evolution — renaming core concepts, modernizing to Java 25, and adding MCP client capabilities.

<CardGrid stagger>
  <Card title="Agent-Centric Architecture" icon="approve-check-circle">
    Core terminology refactored: **Bot → Agent**, **Package → Workflow**, **Behavior → Rules**. The entire codebase and API surface now uses agent-centric language.
  </Card>
  <Card title="39 MCP Tools" icon="approve-check-circle">
    Expanded from 33 to 39 tools — adding 6 schedule management tools for cron-based automation: create, read, update, delete, deploy, and undeploy schedules.
  </Card>
  <Card title="MCP Client" icon="approve-check-circle">
    Agents can now **consume external MCP tools** from other servers via StreamableHTTP transport. Configure external MCP servers in your agent's LLM config.
  </Card>
  <Card title="Java 25 Modernization" icon="approve-check-circle">
    Leveraging unnamed variables (`_`), `toList()` collectors, `instanceof` pattern matching, and enhanced switch expressions throughout the codebase.
  </Card>
  <Card title="Quarkus 3.32.4" icon="approve-check-circle">
    Latest Quarkus release with improved Virtual Threads support, faster native compilation, and enhanced CDI lifecycle management.
  </Card>
  <Card title="Code Quality" icon="approve-check-circle">
    Checkstyle warnings reduced from 697 to 81. Over 1,000 tests with zero failures. Comprehensive MCP tool test coverage.
  </Card>
</CardGrid>

For full details, see the [v6.0 Changelog](/docs/changelog/v6/).
