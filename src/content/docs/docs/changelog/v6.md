## EDDI v6.0 — Enterprise AI Orchestrator

This is a major release that establishes EDDI as a **full-stack enterprise AI orchestration platform**.

### Highlights

- **39 MCP tools** — Full Model Context Protocol server with conversation, admin, resource CRUD, diagnostic, and setup tools
- **MCP Client** — agents can consume external MCP tools via LangChain4j
- **Agent Discovery & Managed Conversations** — Intent-based routing with auto-created conversations
- **Dual Database Support** — MongoDB and PostgreSQL (via database abstraction layer)
- **Security Hardening** — Path traversal fixes, URL validation, vault integration
- **810+ Tests** — Comprehensive test coverage across all MCP tools and core features
- **Sentiment Analysis** — Optional sentiment monitoring with score, trend, emotions, intent, urgency
- **OpenAPI agent creation** — `create_api_agent` parses OpenAPI specs and generates full pipeline configurations

For the complete changelog, see the [CHANGELOG.md on GitHub](https://github.com/labsai/EDDI/blob/main/CHANGELOG.md).
