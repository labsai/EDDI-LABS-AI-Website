EDDI is built on proven open-source foundations — LangChain4j, Quarkus, and the Model Context Protocol — making it compatible with a wide ecosystem of AI tools and infrastructure.

## LLM Providers

EDDI supports **20+ LLM providers** through LangChain4j. Switch between providers by changing a JSON configuration — no code changes required.
